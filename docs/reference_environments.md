# Reference Environments

The default install uses `requirements.txt`, which keeps the public setup
focused on the direct dependencies needed by the metric scripts. The freezes
below record the exact working environments used during the paper experiments.
They are kept as references because they include transitive dependencies,
platform-specific CUDA packages, and two environments with different pins for
packages such as `torch`, `numpy`, `opencv-python`, and `pandas`.
